Lets say we want to look at the relationship between flight delays and the temperature. Maybe colder days have more delays? But unfortunately, our temperature data for each airport (LGA, JFK, and EWR) is stored in separate columns, like below:

flights_temp

In order to generate a plot with average temperature on the x-axis (like we did with wind gust a few questions back), we need to transform our data frame. What is the first thing we need to do?

flights_temp %>% pivot_longer(cols = c("LGA", "JFK", "EWR"), names_to = c("origin"), values_to = "temp")
flights_temp %>% mutate(origin = c("LGA", "JFK", "EWR"),
                        temp = values) 
flights_temp %>% mutate(temp = bind.rows(cols = c("LGA", "JFK", "EWR")) %>% arrange(month, day)
flights_temp %>% bind.rows(cols = c("LGA", "JFK", "EWR")) %>% names_to = "origin" %>% arrange(month, day)
