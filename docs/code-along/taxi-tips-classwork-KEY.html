Your turn!

Create a workflow for a logistic regression model

Make sure you swap out tree_* for logi_* when saving each set of code!

```{r}
# Your code here!

# Step 1: Specify the model with desired parameters
# Here, create a logistic regression model specification
# Set the mode to "classification" since we're predicting a binary outcome
logi_spec <- logistic_reg() %>%
  set_engine("glm") %>%
  set_mode("classification")

# Step 2: Build the workflow
# Initialize a workflow object
# Add the formula (e.g., tip ~ .) to specify the target and predictors
# Attach the logistic regression model specified in Step 1
logi_wflow <- workflow() %>%
  add_formula(tip ~ .) %>%
  add_model(logi_spec)

# Step 3: Fit the workflow to training data
# Use the fit() function to apply the workflow to the training data (e.g., taxi_train)
logi_fit <- logi_wflow %>%
  fit(data = taxi_train)
```

```{r}
# Your code here!

# Create 10-fold cross-validation splits
set.seed(123) # for reproducibility
folds <- vfold_cv(taxi_train, v = 10)

# Refit the logistic regression workflow with cross-validation
logi_resamples <- fit_resamples(
  logi_wflow,         # your logistic regression workflow
  resamples = folds,  # the cross-validation splits
  metrics = metric_set(accuracy, sensitivity, specificity, roc_auc),  # specify metrics you want to evaluate
  control = control_resamples(save_pred = TRUE)  # Save predictions
)
```
